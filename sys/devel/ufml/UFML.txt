UFS / FFS / MFS / LFS (UFML FS): A Modular Super Filesystem 
- Archive, Content Address Based
- Checksums (on Archive)
- Encryption
- Compression
- Snapshots

LOFS/NULLFS: (Stack Filesystem)
UFML: (Base on LOFS/NULLFS)
- UFML Optional
- Implement lower & upper layers in UFML
	- Allow for stacked mounts
	- Lower Layer = UFS/FFS/MFS/LFS (Only)
	- Upper Layer = UFML
- Mounts designated file location (predefined or set)
	- Must be an existing UFS/FFS/MFS/LFS filesystem
